ESP32 Pokakus
=============

ESP32-C3 firmware:

- Connects to WiFi
- Waits for a button click
- Sends a Telegram message

Configuration:

- WiFi SSID & Password
- Telegram bot password
- User id / Group id to send the message to
- Message content

Development
-----------

Use vscode + devcontainer.

Run build:

```console
$ cargo run
```
